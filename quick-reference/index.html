    Quick Reference Library

    Base path¶

    www.recordstackoverflow.com/api/v1/

    Records¶

    HTTP Method URL Description
    POST /records/ Create a new record
    GET /records/ Retrieve all records
    GET /records/merchant/{merchantId}/ Retrieve all records by merchant
    PUT /records/{recordId}/ Update a record
    DELETE /records/{recordId}/merchants/{merchantId}/ Delete a record

    Orders¶

    HTTP Method URL Description
    POST /orders/merchants/{merchantId}/records/{recordId}/ Create and order
    GET /orders/customers/{customerId}/ Retrieve all orders for a customer
    GET /orders/merchants/{merchantId}/ Retrieve all orders for a merchant
    PUT /orders/{orderId}/ Update an order by ID
    DELETE /orders/{orderId}/customers/{customerId}/ Delete an order

    Merchants¶

    HTTP Method URL Description
    GET /merchants/{merchantId}/ Retrieve merchant information
    PUT /merchants/{merchantId}/ Update a merchant by ID

    Customers¶

    HTTP Method URL Description
    GET /customers/{customerId}/ Retrieve customer information
    PUT /customers/{customerId}/ Update a customer by ID
